# Hướng Dẫn Cài Đặt Ứng Dụng
- Bước 1: mở terminal vào thư mục project hiện tại
- Bước 2: cd docker
- Bước 3: Mở docker desktop
- Bước 4: Chạy lệnh sh .\pull.sh
- Bước 5: Chạy lệnh sh .\run.sh
- Bước 6: Chạy file main.dart trong ứng dụng
